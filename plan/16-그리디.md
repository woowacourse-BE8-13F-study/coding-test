# 16. 그리디

## 📚 개념 정리 (문제 풀기 전 필독)

> 교재 목차 소제목: 그리디 개념·거스름돈 · 최소 신장 트리(신장 트리·MST) · 배낭 문제(부분 배낭·0/1 배낭)

- 그리디는 매 순간 지역적으로 최선인 선택을 하여 전체 최적해를 노리는 기법이다.
- '탐욕적 선택 속성'과 '최적 부분 구조'가 성립할 때만 최적해가 보장된다.
- 거스름돈, 최소 신장 트리(크루스칼/프림), 부분 배낭 문제가 대표적이다.

### 꼭 알아야 할 선행 지식
- 정렬 후 탐욕 선택
- MST(크루스칼·유니온파인드)
- 부분 vs 0/1 배낭 구분

### 핵심 연산의 시간복잡도

| 연산 | 시간복잡도 |
|---|---|
| 정렬 기반 그리디 | O(n log n) |
| 크루스칼 | O(E log E) |
| 부분 배낭(정렬) | O(n log n) |
| 0/1 배낭 | DP 필요 O(nW) |

### 자바 구현 포인트
- 대부분 '정렬 후 순차 선택' 패턴 → 비교자 설계가 핵심.
- MST는 크루스칼(간선 정렬+유니온파인드) 또는 프림(우선순위 큐).
- 부분 배낭은 그리디로 풀리지만 0/1 배낭은 DP가 필요함에 주의.

### 자주 하는 실수 / 주의점
- 그리디로 안 풀리는 문제(0/1 배낭)에 그리디 적용
- 선택 기준(정렬 키) 오류
- 반례 검토 없이 그리디 확신
