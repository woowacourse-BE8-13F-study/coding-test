# 00. 오리엔테이션 — 스터디 방법 안내

## 이 스터디는?
- 교재: **코딩 테스트 합격자 되기: 자바 편** (골든래빗)
- 범위: **05장 배열 ~ 17장 코딩 테스트 모의고사**
- 플랫폼: **프로그래머스** (모든 문제를 프로그래머스에서 풉니다)
- 기간: **2026-07-22 ~ 2026-09-04** (우테코 레벨 3 기간)
- 하루 목표: **평일 약 2시간 / 주말 약 1시간**

> 예산은 명세 기본값(평일 1시간)보다 상향되어 있습니다. 교재 05~17장 전 범위를 마감일까지 모두 소화하기 위한 조정입니다. 부담되면 평일 1시간까지만 하고 나머지는 다음 날로 미뤄도 됩니다.

## 하루 학습 흐름
1. 그날 날짜의 [일자별 계획](../schedule/일자별-계획.md)을 엽니다.
2. 파트가 처음 시작되는 날이면 해당 [파트 문서](.)의 **📚 개념 정리**를 먼저 읽습니다.
3. 배정된 문제를 **[문제 열기]** 링크로 프로그래머스에서 열어 풉니다.
4. 풀이를 `solutions/`에 올리고 체크박스를 체크합니다. ([참여 방법](../CONTRIBUTING.md))

## 난이도(별점) 표기
| 별점 | 의미 | 부하 |
|---|---|---|
| ★ | 쉬움 | 1 |
| ★★ | 보통 | 2 |
| ★★★ | 어려움 | 3 |
| ★★★★ | 매우 어려움 | 4 |
| ★★★★★ | 최상 | 5 |

- **부하로 분량을 판단**합니다. 쉬운 문제는 여러 개, 어려운 문제는 하나만/이틀에 나눠 풉니다.
- `(1/2)`,`(2/2)` 표기는 **한 문제를 이틀에 나눠** 푸는 고난도 문제입니다.
- `⚠️확인필요` 표기는 교재 목차 OCR 기준이라 **문제명/난이도/회차가 불확실**한 항목입니다. 프로그래머스에서 확인 후 보정하세요.
