# 00. 오리엔테이션 — 스터디 방법 안내

## 이 스터디는?
- 교재: **코딩 테스트 합격자 되기: 자바 편** (골든래빗)
- 범위: **05장 배열 ~ 17장 코딩 테스트 모의고사**
- 플랫폼: **프로그래머스** (몸풀기 문제 포함 모든 문제를 프로그래머스에서 풉니다)
- 기간: **2026-07-22 ~ 2026-09-04** (우테코 레벨 3 기간)
- 하루 목표: **평일 약 2시간 / 주말 약 1시간**
- 진행 방식: **스터디원 전원 같은 진도**. 각자 푼 풀이는 개인 브랜치에 올리고 `main`으로 PR.

## 하루 학습 흐름
1. [README의 학습 캘린더](../README.md#-학습-캘린더)에서 **오늘 날짜**를 클릭합니다.
2. 그날 항목에 개념 세션(📚)이 있으면 해당 [파트 문서](.)의 **개념 정리**를 먼저 읽습니다.
3. 배정된 문제를 **[문제 열기]** 링크로 프로그래머스에서 열어 풉니다.
4. 풀이를 개인 브랜치에 커밋 → `main`으로 PR → 머지. 체크박스도 함께 체크합니다. ([참여 방법](../CONTRIBUTING.md))

## 난이도(별점) 표기
| 별점 | 의미 |
|---|---|
| ★ | 쉬움 |
| ★★ | 보통 |
| ★★★ | 어려움 |
| ★★★★ | 매우 어려움 |
| ★★★★★ | 최상 |

- **난이도로 분량을 판단**합니다. 쉬운 문제는 여러 개, 어려운 문제는 하나만/이틀에 나눠 풉니다.
- `(1/2)`,`(2/2)` 표기는 **한 문제를 이틀에 나눠** 푸는 고난도 문제입니다.
- `⚠️확인필요` 표기는 교재 목차 OCR 기준이라 **문제명/난이도/회차가 불확실**한 항목입니다. 프로그래머스에서 확인 후 보정하세요.

## 몸풀기 문제 안내
- 교재의 몸풀기 문제 중 일부는 저자 출제 문제라 프로그래머스에 없습니다. 그래서 **같은 주제의 실제 프로그래머스 문제(난이도 낮음)**로 교체했습니다.

## 링크 규칙
- 모든 문제 링크는 프로그래머스 **검색 링크**입니다(항상 동작). 검색 결과에서 해당 문제를 클릭해 여세요.
- 도메인: `https://school.programmers.co.kr`
