# 15. 동적 계획법

## 📚 개념 정리 (문제 풀기 전 필독)

> 교재 목차 소제목: 점화식 세우기와 DP · 메모이제이션 · 최장 증가 부분 수열(LIS) · 최장 공통 부분 수열(LCS)

- 동적 계획법(DP)은 큰 문제를 겹치는 부분 문제로 나누고, 결과를 저장(메모이제이션)해 재계산을 피한다.
- 핵심은 상태 정의와 점화식 세우기다. 하향식(재귀+메모)과 상향식(반복+테이블) 두 방식이 있다.
- LIS·LCS·배낭 등은 반드시 익혀야 할 대표 DP 패턴이다.

### 꼭 알아야 할 선행 지식
- 재귀와 부분 문제 중복
- 점화식 세우기
- 메모이제이션/타뷸레이션

### 핵심 연산의 시간복잡도

| 연산 | 시간복잡도 |
|---|---|
| 피보나치(DP) | O(n) |
| LIS(기본) | O(n²) |
| LIS(이분탐색) | O(n log n) |
| LCS | O(n·m) |

### 자바 구현 포인트
- 메모 배열은 '방문 안 함' 표시(예: -1)로 초기화 후 재사용.
- 상태가 2개면 `dp[i][j]`, 공간 최적화 시 롤링 배열 고려.
- 점화식을 주석으로 먼저 적고 코드로 옮기면 실수가 준다.

### 자주 하는 실수 / 주의점
- 상태 정의 모호로 점화식 실패
- 기저 조건 누락
- 메모 초기값과 실제 값 충돌
