# 13. 정렬

## 📚 개념 정리 (문제 풀기 전 필독)

> 교재 목차 소제목: 정렬 개념 — 버블·삽입·병합·힙·계수 정렬 및 시간복잡도 비교

- 정렬은 데이터를 규칙에 맞게 나열하는 것으로, 비교 기반 정렬의 하한은 O(n log n)이다.
- 버블·삽입은 O(n²), 병합·힙은 O(n log n), 계수 정렬은 값 범위가 작을 때 O(n+k)로 동작한다.
- 실전에서는 언어 기본 정렬(자바 Dual-Pivot Quicksort/TimSort)을 쓰고 비교자만 잘 정의하면 된다.

### 꼭 알아야 할 선행 지식
- 안정 정렬 개념
- 비교자(Comparator)
- 계수 정렬 조건(값 범위)

### 핵심 연산의 시간복잡도

| 연산 | 시간복잡도 |
|---|---|
| 버블/삽입/선택 | O(n²) |
| 병합/힙 | O(n log n) |
| 계수 정렬 | O(n+k) |
| 자바 기본 정렬 | O(n log n) |

### 자바 구현 포인트
- 기본형 배열: `Arrays.sort` (Dual-Pivot Quicksort), 객체 배열/리스트: TimSort(안정).
- 커스텀 정렬: `Arrays.sort(a, (x,y) -> ...)` 또는 `list.sort(Comparator...)`.
- 다중 기준: `Comparator.comparingInt(...).thenComparing(...)`.

### 자주 하는 실수 / 주의점
- Comparator 대소 반환값 부호 실수
- int 뺄셈 비교의 오버플로우
- 정렬 안정성 필요한데 불안정 정렬 사용
