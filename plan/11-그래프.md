# 11. 그래프

## 📚 개념 정리 (문제 풀기 전 필독)

> 교재 목차 소제목: 그래프 개념·용어·종류·구현 · DFS · BFS · DFS·BFS 비교 · 다익스트라 · 벨만-포드

- 그래프는 정점과 간선으로 관계를 표현하며, 방향/무방향·가중치 유무로 종류가 나뉜다.
- DFS는 스택/재귀로 깊게, BFS는 큐로 넓게 탐색하며 BFS는 가중치 없는 최단거리에 쓰인다.
- 가중치 최단경로는 다익스트라(음수 간선 없음)·벨만-포드(음수 허용)로 푼다.

### 꼭 알아야 할 선행 지식
- 인접 리스트/인접 행렬 표현
- 방문 배열(visited)
- 우선순위 큐(다익스트라)

### 핵심 연산의 시간복잡도

| 연산 | 시간복잡도 |
|---|---|
| DFS/BFS | O(V+E) |
| 다익스트라(PQ) | O(E log V) |
| 벨만-포드 | O(V·E) |
| 인접 행렬 공간 | O(V²) |

### 자바 구현 포인트
- 인접 리스트: `List<List<int[]>> adj;` (가중치는 `int[]{to,w}`).
- BFS: `Queue` + `visited[]`, 최단거리 문제의 기본기.
- 다익스트라: `PriorityQueue`로 최소 거리 정점을 꺼내며 갱신(`dist[]`).

### 자주 하는 실수 / 주의점
- 방문 처리 타이밍 오류로 무한 루프/중복 방문
- 무방향 그래프에서 양방향 간선 누락
- 다익스트라에 음수 간선 사용
