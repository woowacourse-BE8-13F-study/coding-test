# 14. 시뮬레이션

## 📚 개념 정리 (문제 풀기 전 필독)

> 교재 목차 소제목: 시뮬레이션 풀이 노하우 · 행렬 연산 · 좌표 연산 · 대칭·회전 연산

- 시뮬레이션은 문제에 서술된 규칙을 그대로 코드로 옮겨 상태 변화를 따라가는 유형이다.
- 격자/좌표 위 이동, 회전·대칭 같은 행렬 연산을 정확히 구현하는 것이 관건이다.
- 알고리즘적 통찰보다 '조건을 빠짐없이 구현'하는 꼼꼼함이 더 중요하다.

### 꼭 알아야 할 선행 지식
- 2차원 배열 좌표 다루기
- 방향 벡터(dx/dy)
- 행렬 회전/전치/대칭

### 핵심 연산의 시간복잡도

| 연산 | 시간복잡도 |
|---|---|
| 격자 순회 | O(행×열) |
| 회전/전치 | O(n²) |
| 좌표 이동 시뮬 | O(단계 수) |
| 경계 검사 | O(1) |

### 자바 구현 포인트
- 방향 배열: `int[] dx={0,0,1,-1}, dy={1,-1,0,0};` 로 상하좌우 이동.
- 90도 회전: `dst[j][n-1-i] = src[i][j]`, 전치: `dst[j][i]=src[i][j]`.
- 경계 검사 헬퍼(`inRange`)를 만들어 중복 조건을 정리.

### 자주 하는 실수 / 주의점
- 행/열과 x/y 축 혼동
- 회전 방향/공식 실수
- 경계 밖 접근
