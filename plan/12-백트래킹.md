# 12. 백트래킹

## 📚 개념 정리 (문제 풀기 전 필독)

> 교재 목차 소제목: 백트래킹·백트래킹 알고리즘 개념 · 유망 함수 · 4-퀸 문제

- 백트래킹은 해를 하나씩 만들어 보다가 조건을 위반하면 되돌아가는(가지치기) 완전탐색 기법이다.
- 유망 함수(promising)로 더 진행할 가치가 없는 경로를 미리 잘라 탐색량을 줄인다.
- 순열·조합·N-퀸·스도쿠처럼 '경우의 수를 체계적으로 탐색'하는 문제에 쓰인다.

### 꼭 알아야 할 선행 지식
- 재귀
- 상태 되돌리기(visited 원복)
- 가지치기(유망 함수)

### 핵심 연산의 시간복잡도

| 연산 | 시간복잡도 |
|---|---|
| 최악(가지치기 없음) | 지수 시간 |
| N-퀸 | O(N!) 근처 |
| 가지치기 효과 | 실전에서 크게 감소 |
| 깊이 | O(선택 단계 수) |

### 자바 구현 포인트
- 재귀 함수에 현재 상태(깊이, 부분해)를 넘기고, 선택→재귀→선택 취소 패턴.
- 방문/사용 표시는 배열로, 재귀 후 반드시 원복.
- 가능한 한 이른 시점에 조건 검사로 가지치기.

### 자주 하는 실수 / 주의점
- 상태 원복 누락으로 오답
- 종료 조건 오류로 무한 재귀
- 중복 조합/순열 처리 미흡
