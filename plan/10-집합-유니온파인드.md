# 10. 집합 (유니온-파인드)

## 📚 개념 정리 (문제 풀기 전 필독)

> 교재 목차 소제목: 집합 개념 · 상호배타적 집합 · 배열 트리로 집합 표현 · 유니온-파인드 알고리즘

- 서로소 집합(Disjoint Set)은 겹치지 않는 그룹들을 관리하는 자료구조다.
- 유니온-파인드는 find(대표 찾기)와 union(두 집합 합치기)로 '같은 그룹인가'를 빠르게 판단한다.
- 경로 압축 + 랭크(크기) 기반 합치기를 쓰면 거의 상수 시간에 동작한다.

### 꼭 알아야 할 선행 지식
- 배열로 부모 표현(parent[])
- 재귀/반복 find
- 그래프 연결성과의 관계

### 핵심 연산의 시간복잡도

| 연산 | 시간복잡도 |
|---|---|
| find(경로압축) | ≈ O(α(n)) |
| union | ≈ O(α(n)) |
| 초기화 | O(n) |
| 최적화 없이 | O(n) |

### 자바 구현 포인트
- `int[] parent;` 초기화는 `parent[i]=i`.
- find: `return parent[x]==x ? x : (parent[x]=find(parent[x]));` (경로 압축).
- union: 두 루트를 찾아 한쪽을 다른쪽 부모로; 랭크/크기 배열로 균형 유지.

### 자주 하는 실수 / 주의점
- 경로 압축 누락으로 성능 저하
- union 전에 루트를 안 찾고 값 자체를 이어붙임
- 1-based/0-based 인덱스 혼동
