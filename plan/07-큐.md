# 07. 큐

## 📚 개념 정리 (문제 풀기 전 필독)

> 교재 목차 소제목: 큐 개념 · 데이터 이동 과정 · 활용 분야 · 큐 ADT

- 큐는 선입선출(FIFO) 자료구조로, 먼저 넣은 데이터가 먼저 나온다.
- BFS, 작업 대기열, 시뮬레이션의 순서 처리에 사용된다.
- enqueue/dequeue 는 상수 시간이며, 우선순위가 필요하면 우선순위 큐를 쓴다.

### 꼭 알아야 할 선행 지식
- FIFO 개념
- 원형 큐(링 버퍼)
- 우선순위 큐/힙 개념 맛보기

### 핵심 연산의 시간복잡도

| 연산 | 시간복잡도 |
|---|---|
| enqueue(offer) | O(1) |
| dequeue(poll) | O(1) |
| peek | O(1) |
| 탐색 | O(n) |

### 자바 구현 포인트
- `Queue<Integer> q = new ArrayDeque<>();` 또는 `LinkedList`.
- 삽입=`offer`, 제거=`poll`, 확인=`peek` (예외 대신 null/false 반환하는 쪽 권장).
- 우선순위 큐: `PriorityQueue<>` (기본 최소 힙, 비교자로 최대 힙 구성).

### 자주 하는 실수 / 주의점
- `poll`이 비었을 때 null 반환 → NPE 주의
- 큐/스택 방향 혼동
- 우선순위 큐를 일반 큐로 오해
