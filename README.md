# 🧑‍💻 코딩 테스트 스터디 — 합격자 되기(자바 편)

## 📖 스터디 소개
- **교재:** 코딩 테스트 합격자 되기: 자바 편 (골든래빗)
- **범위:** 05장 배열 ~ 17장 코딩 테스트 모의고사
- **기간:** 2026-07-22(수) ~ 2026-09-04(금)
- **하루 목표:** 평일 약 1시간 · 주말 약 30분
- **플랫폼:** [프로그래머스](https://school.programmers.co.kr)
- **진행 방식:** 스터디원 전원 **동일한 진도**로 진행하고, 각자 푼 풀이를 **개인 브랜치 → `main` PR**로 올립니다. ([참여 방법](CONTRIBUTING.md))

## ✅ 규칙 요약
1. 모든 문제는 **프로그래머스**로 연결합니다. (몸풀기 문제도 실제 프로그래머스 문제로 구성)
2. 아래 **학습 캘린더**에서 날짜를 클릭하면 그날 풀 문제만 모아서 볼 수 있습니다.
3. 하루 약 1~2시간, **난이도(별점)로 분량을 판단**합니다.
4. 각 파트는 **개념 세션(📚)을 먼저** 보고 문제를 풉니다.
5. 시작일부터 마감일까지 **날짜별로** 학습합니다.
6. **주말도 진행하되 가볍게**(복습·쉬운 문제 위주).
